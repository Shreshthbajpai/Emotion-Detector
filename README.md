# 😃 Emotion Detector

This project is an AI-powered **Emotion Detection** system that detects human facial emotions in real-time using a webcam. It uses computer vision and deep learning to classify emotions such as:

- **Happy**
- **Sad**
- **Angry**
- **Surprised**
- **Neutral**
