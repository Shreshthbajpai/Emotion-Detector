# 😃 Emotion Detector

This is a real-time **Emotion Detection** system using deep learning and computer vision. The model detects facial expressions from webcam input and classifies emotions such as:

- 😃 Happy  
- 😢 Sad  
- 😠 Angry  
- 😲 Surprise  
- 😐 Neutral
